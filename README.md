# Data Analytics Projects

Welcome to my data analytics portfolio. This repository contains Python-based analysis projects focused on cleaning, processing, and visualizing large datasets.

## Projects

### 1. Maritime AIS Data Analysis (Jan 2025)
* **Tools:** Python, Pandas, Matplotlib, Zstandard.
* **Goal:** Analyzed NOAA maritime traffic data to identify peak vessel activity hours and calculated engine active time for specific vessels.
* **Key Finding:** Identified peak maritime traffic at 18:00 UTC and calculated a total of 8 hours of active engine time for the Henry Hudson.

### 2. NYC 311 Service Requests Dashboard
* **Tools:** Tableau, Data Cleaning.
* **Goal:** Visualized city-wide service requests to identify trends in municipal issues.
